TrackStories Studio

Minimaler Editor für Strecken, Sender, Quiz und Rewards. Änderungen werden erst übernommen, wenn du im jeweiligen Bereich auf „Speichern“ klickst — beim Durchklicken wird nichts überschrieben.

Übersicht

Hier siehst du schnell, was in der gewählten Strecke enthalten ist.

Strecke

Basisdaten, Startscreen und Onboarding.

Dieser Text wird beim Start der Reise vorgelesen. Wenn leer, baut die App den Intro-Text automatisch aus Strecke, Sitzseite und Sender.

Sender

Sender durchklicken, bearbeiten, speichern oder löschen. Neue Sender starten leer.

Quiz

Ein Quiz gehört über die ID zu einem Sender. Quiz kann jetzt gezielt eine Reward-ID auslösen. So bleibt klar, welche Belohnung zu welcher Frage gehört.

Rewards

Sammelobjekte und Badges. Die ID wird im Quiz als Reward-ID ausgewählt und dadurch eindeutig ausgelöst.

JSON

Für präzise Änderungen.

Export

App exportieren oder Template aktualisieren.
